# CCS811 library
Arduino library for itbrainpower.net CCS811 I2C sensor breakout. CCS811 it's an TVOC/eCO2 Sensor by AMS.

Compatible with:
	s-Sense CCS811 I2C sensor breakout [PN: SS-CCS811#I2C, SKU: ITBP-6004], info https://itbrainpower.net/sensors/CCS811-CO2-TVOC-I2C-sensor-breakout 
	s-Sense CCS811 + HDC2010 I2C sensor breakout [PN: SS-HDC2010+CCS811#I2C, SKU: ITBP-6006], info https://itbrainpower.net/sensors/CCS811-HDC2010-CO2-TVOC-TEMPERATURE-HUMIDITY-I2C-sensor-breakout

For CCS811 + HDC2010 combo sensors, the CCS811 temperature and humidity calibration can be done using data gathered from integrated HDC2010 sensor.

More info about itbrainpower.net sensors can be foud here: https://itbrainpower.net/sensors
